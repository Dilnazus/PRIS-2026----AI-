Описание проекта  
В данной лабораторной работе реализован простой игровой искусственный интеллект.
Игровой AI управляет ботом, который взаимодействует с пользователем в мини-игре
и принимает решения на основе заданных правил.

Цель работы  
Изучение принципов проектирования игровых интеллектуальных систем и
архитектуры взаимодействия пользователя и игрового AI.

Задачи работы  
Инициализация проекта и настройка рабочего окружения.  
Реализация простого игрового AI.  
Создание пользовательского интерфейса для взаимодействия с AI.  
Описание архитектуры системы.

Используемые технологии  
Python  
Streamlit  
GitHub
